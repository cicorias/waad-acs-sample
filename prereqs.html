    • To deploy and run this sample you will need the following:

    • A Windows Azure subscription for your Windows Azure Active Directory domain, your Access Control Services namespace, and to host your ASP.NET MVC website. You can sign up for a free trial subscription here: Windows Azure Free trial
    • Windows Azure Active Directory Module for Windows PowerShell cmdlets http://aka.ms/aadposh

    Installed using the Web Platform Installer:

    • Windows Azure Tools for Visual Studio (latest version)
    • Windows Azure SDK (latest version)
    • Windows Azure PowerShell (latest version)
    • Windows Azure Libraries (latest version)
    • Windows Identity Foundation
    • IIS Express

    This are just the Windows Azure related pieces. You'll also need the standard libraries/tools/SDKs for ASP.NET MVC web site development.
