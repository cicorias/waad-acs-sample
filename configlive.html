    If the site is going to be able to retrieve user information from the Live servers when a user authenticates using Microsoft Account (Live) credentials, then you must configure a Live application at Live Connect Developer Center.

    This screenshot shows an example application:

    You can use any name you like for the application. The Client ID and Client secret are generated for you and you will need these when you configure the web application. The Redirect domain should be the domain name you use for the application. If you are running the application locally, you will need to edit you hosts file to use this domain. See Configuring the Website for more information.
